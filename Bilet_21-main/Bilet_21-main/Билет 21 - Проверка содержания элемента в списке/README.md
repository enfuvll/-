# Билет 21 - Проверка содержания элемента в списке

## Tasks

1. Проверка содержания элемента в списке
2. Объединение коллекций
3. WinForms mini application

## Files

- `TicketTasks.cs` - functions for both tasks.
- `Tests.cs` - Assert checks, started before the form opens.
- `MainForm.cs` - WinForms interface for manual checks.

Open `Ticket21.csproj` in Visual Studio, or open the root `AllTickets.sln`.
