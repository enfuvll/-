# Билет 09 - Абсолютное значение отрицательного числа

## Tasks

1. Абсолютное значение отрицательного числа
2. Поиск строк по началу
3. WinForms mini application

## Files

- `TicketTasks.cs` - functions for both tasks.
- `Tests.cs` - Assert checks, started before the form opens.
- `MainForm.cs` - WinForms interface for manual checks.

Open `Ticket09.csproj` in Visual Studio, or open the root `AllTickets.sln`.
